# 人工智能：现代方法 - 完整案例集合

## 项目简介

本项目是基于 Stuart Russell 和 Peter Norvig 的经典教科书《人工智能：现代方法》(Artificial Intelligence: A Modern Approach)的完整案例集合。通过理论讲解、实际案例和代码实现，帮助读者深入理解人工智能的核心概念和技术。

## 主要特色

### 1. 完整覆盖

- 涵盖书中所有核心章节
- 从基础概念到高级应用
- 理论与实践相结合

### 2. 实际案例

- 每个章节都有具体的应用案例
- 真实世界的问题解决方案
- 可执行的代码实现

### 3. 渐进式学习

- 从简单到复杂的学习路径
- 每个概念都有详细解释
- 充分的练习和实验

### 4. 现代技术

- 使用最新的 Python 库和框架
- 结合深度学习和传统 AI 方法
- 可视化和交互式演示

## 技术栈

- **编程语言**: Python 3.8+
- **核心库**: NumPy, Pandas, Scikit-learn
- **深度学习**: TensorFlow, PyTorch
- **可视化**: Matplotlib, Seaborn, Plotly
- **自然语言处理**: NLTK, spaCy
- **强化学习**: OpenAI Gym
- **图形算法**: NetworkX

## 快速开始

### 1. 环境设置

```bash
# 克隆项目
git clone <repository-url>
cd artificial-intelligence-guide

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate     # Windows

# 安装依赖
pip install -r requirements.txt
```

### 2. 运行示例

```bash
# 运行智能代理示例
python 01-智能代理/案例实现/simple_agent.py

# 运行搜索算法示例
python 02-问题求解/案例实现/search_algorithms.py

# 运行机器学习示例
python 05-机器学习/案例实现/decision_tree.py
```

### 3. Jupyter Notebook

许多案例都提供了 Jupyter Notebook 版本，可以交互式学习：

```bash
jupyter notebook
```

## 学习路径建议

### 初学者路径

1. **智能代理基础** (第 1 部分)
2. **搜索算法** (第 2 部分 - 搜索相关)
3. **机器学习入门** (第 5 部分 - 决策树)
4. **实际项目** (项目案例)

### 进阶路径

1. **逻辑和推理** (第 3 部分)
2. **概率和不确定性** (第 4 部分)
3. **高级机器学习** (第 5 部分 - 神经网络、强化学习)
4. **自然语言处理** (第 6 部分 - NLP 相关)
